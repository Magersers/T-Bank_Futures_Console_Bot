# T-Bank Futures Console Bot

Консольный Python-бот для T-Bank Invest API (бывш. Тинькофф Инвестиции), который:

- Открывает LONG и/или SHORT одновременно.
- Делит общий объем лотов на сторону максимум на **3 независимые сделки**.
  - Пример: `6` лотов в LONG = `3` сделки по `2` лота.
- Целевой выход по прибыли: **минимум +0.06%** (или выше, если рынок дает).
- Стоп-лосс: **-0.2%**, после чего включается пауза новых входов на **2 минуты**.
- Кэширует токен/портфель/figi/лимиты в `~/.tbank_futures_bot/config.json`.
- Получает данные стакана через **MarketData stream** (bidirectional stream), одним потоком без polling.

## Важно

- Это **не** гарантия дохода. На реальном рынке возможны проскальзывания, комиссии и убытки.
- Перед реальной торговлей протестируй на небольших объемах/песочнице.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск

```bash
python bot.py
```

При запуске бот спросит:

1. токен API,
2. `account_id` (портфель),
3. FIGI инструмента,
4. общий объем лотов LONG (0..N),
5. общий объем лотов SHORT (0..N).

## Логика

- При старте объем на сторону разбивается на максимум 3 независимые сделки.
  - `1` -> `[1]`
  - `2` -> `[1, 1]`
  - `3` -> `[1, 1, 1]`
  - `6` -> `[2, 2, 2]`
  - `7` -> `[3, 2, 2]`
- Цена берется из потокового стакана (`market_data_stream`, `depth=1`), вычисляется mid-price.
- При наличии свободного слота бот открывает недостающие сделки по плану.
- Для каждой позиции:
  - LONG: TP при `price >= entry * 1.0006`, SL при `price <= entry * 0.998`.
  - SHORT: TP при `price <= entry * 0.9994`, SL при `price >= entry * 1.002`.
- В консоль выводится обратная связь по закрытию сделки: `qty`, валовый PnL, комиссия `0.05%`, итоговый (net) PnL и метка `ПРИБЫЛЬ/УБЫТОК`.
- При SL по любой позиции бот закрывает ее и уходит в cooldown 120 секунд (новые входы запрещены).
